Listas

Listas desordenadas

Una lista desordenada en HTML es un conjunto de elementos agrupados sin un orden específico, representados comúnmente con viñetas. Se define mediante la etiqueta <ul> (unordered list), y cada ítem dentro de la lista se encapsula con la etiqueta <li> (list item). Este tipo de lista es ideal para presentar información que no requiere una secuencia lógica, como características de un producto, ingredientes de una receta o enlaces de navegación. Su estructura semántica contribuye a la accesibilidad y organización del contenido en la web.

Listas ordenada

Una lista ordenada en HTML se utiliza para presentar elementos que siguen una secuencia lógica o jerárquica. Se define con la etiqueta <ol> (ordered list), y cada elemento de la lista se encapsula dentro de una etiqueta <li>. Al renderizarse en el navegador, los ítems aparecen numerados automáticamente, lo que facilita la lectura de pasos, instrucciones o clasificaciones. Este tipo de lista es especialmente útil cuando el orden de los elementos influye en su significado, como en recetas, procedimientos técnicos o rankings.

Sobre una imagen

Las imágenes en HTML cumplen un rol fundamental en la presentación visual del contenido. No solo enriquecen la experiencia del usuario, sino que también pueden transmitir información clave de forma más rápida y atractiva. Es importante acompañarlas de atributos como alt para mejorar la accesibilidad y utilizar formatos adecuados según el contexto.

HTML

HTML, o Lenguaje de Marcado de Hipertexto, es el estándar utilizado para estructurar el contenido de las páginas web. Permite definir elementos como títulos, párrafos, listas, enlaces e imágenes, organizando la información de forma semántica y accesible para los navegadores.
